# AESTHETIC PROTOCOL: SILENT AUTHORITY

## 1. TONAL LAYERING (THE VOID)
- **Palette:** Deep Blacks and Gunmetal Silvers.
- **Tones:**
  - Background: `#050505`
  - Surface: `#0A0A0A`
  - Elevated: `#111111`
  - Text: `#FFFFFF` (Primary), `#BFC3C7` (Secondary/Silver), `#444444` (Utility).
- **Prohibition:** No vibrant purple/blue gradients (AI Standard). Only neutral, metallic, or mono-tonal gradients.

## 2. OPTICAL TYPOGRAPHY
- **Kerning:** Increase `letter-spacing` for all caps headers by 0.2em - 0.5em.
- **Hierarchy:** Use `clamp()` for responsive font sizing. Never let text overlap.
- **Mix:** High-contrast pairing. Inter (Sans) for structure, JetBrains Mono for technical data.

## 3. GRID SOVEREIGNTY
- **System:** Strict 24-column grid.
- **Gaps:** Large gutters (2rem+) to allow design "to breathe."
- **Overlays:** Static noise and scanline overlays (3-5% opacity) for digital texture.

## 4. COMPONENT DNA
- **Shape:** Sharp edges (0px) or very subtle (2-4px) rounding for technical items.
- **Buttons:** Border-based. Invert on hover.
- **Glass:** High blur (20px+), very low opacity (0.02 - 0.05).
